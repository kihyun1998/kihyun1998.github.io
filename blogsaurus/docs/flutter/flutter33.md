# 33. [flutter] Icon 변경
---

## Windows 기준
---

`windows/runner/Runner.rc`에


```
/////////////////////////////////////////////////////////////////////////////
//
// Icon
//

// Icon with lowest ID value placed first to ensure application icon
// remains consistent on all systems.
IDI_APP_ICON            ICON                    "resources\\app_icon.ico"
```

이런 내용으로 아이콘 경로가 설정돼있어서 변경할 수 있다.
