## NestJs

1. nestjs-cli 설치

```bash
npm i -g @nestjs/cli
```

2. 프로젝트 시작

```bash
nest new
```

원하는 디렉토리가서 `nest new` 하면 된다.

그리고 프로젝트 이름을 정하고 npm 쓸지 yarn쓸지 정해야하는데 원래는 yarn을 좋아하는데 강의 때문에 npm을 선택한 !
