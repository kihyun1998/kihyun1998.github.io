# 해쉬 맵 ( Hash Map )
---

시간복잡도 : O(1)

key-value로 이루어져있다.


할당 방식 : 해쉬 함수를 통해서 값을 구하고 특정 수로 나눈 나머지 값을 통해 위치 할당을 합니다.

여기서 나머지 값이 같다면 linked list를 사용해서 붙입니다.

그래서 탐색할 때 해당 나머지 값 > 찾고자 하는 키값으로 찾습니다.
