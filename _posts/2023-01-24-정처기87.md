# **Section 87.** 관계대수 및 관계 해석
---


## 관계대수란?

관계형 데이터베이스에서 원하는 정보를 검색하기 위해서 사용하는 언어

- 릴레이션을 처리하기 위해 연산자와 연산 규칙을 제공하는 언어
- 피연산자 = 릴레이션, 결과 = 릴레이션
- **순수 관계 연산자** : Select, Project, Join, Division
- **일반 집합 연산자** : UNION(합집합), INTERSECTION(교집합), DIFFERENCE(차집합), CARTESIAN PRODUCT(교차곱)


---


## 순수 관계 연산자


### Select

선택 조건을 만족하는 튜플의 부분 집합을 구하는 연산

- 행에 해당하는 튜플 구하는 연산(**수평 연산**)
- 연산자 기호는 시그마(σ)를 사용

**표기 형식** : σ 조건 {릴레이션 이름}
- 비교 연산이 되고 논리 연산도 된다.
- ex) σ Avg>80 성적


### Project

속성 리스트에 제시된 속성 값만을 추출하여 새 릴레이션을 만드는 연산 (중복은 제거)

- 열에 해당하는 속성을 추출하는 연산(**수직 연산**)
- 연산자 기호는 파이(π)를 사용

**표기 형식** : π 속성 리스트 {릴레이션 이름}
- ex) π Name,Avg 성적


### Join

공통 속성을 중심으로 두 릴레이션을 하나로 합쳐서 새 릴레이션을 만드는 연산

- Join의 결과는 Cartesian Product를 수행한 후 Select를 수행한 것과 같음
- 연산자의 기호는 ⋈를 사용

**표기형식** : {릴레이션 A 이름} ⋈ 키속성(릴레이션 A) = 키속성(릴레이션 B) {릴레이션 B 이름}
- ex) 성적 ⋈ No=No 출석부
- 위 같은 경우에는 자연 조인(Natural Join)도 가능하다.
    - ex) 성적 ⋈ No 출석부

### Division

X는 릴레이션 A의 속성이고 Y는 릴레이션 B의 속성일 때 Y가 X의 부분집합이면 A-B 속성값을 구하는 연산

- 연산자의 기호는 ÷를 사용

**표기형식** : {릴레이션 A 이름} [A 속성 ÷ B 속성] {릴레이션 B 이름}
- ex) 성적 [No, name ÷ No, name] 출석부

## 일반 집합 연산자

수학적 집합 이론에서 사용하는 연산자로 릴레이션 연산에도 사용할 수 있다.

- 합집합, 교집합, 차집합 연산을 하기 위해서는 합병 조건을 만족해야 한다.
> `합병 조건` : 합병하려는 두 릴레이션의 속성 수가 같아야 하고 대응되는 속성의 도메인이 같아야 한다.(이름은 달라도 됨)


### 합집합 (UNION, ∪)

- 튜플의 합집합 구하고 중복되는 튜플은 제거
- **카디널리티** :  두 릴레이션의 카디널리티의 합보다 크지 않음(중복 제거 되어서)

### 교집합 (INTERSECTION, ∩)

- 튜플의 교집합을 구하는 연산
- **카디널리티** : 두 릴레이션 중 카디널리티가 적은 릴레이션의 카디널리티보다 크지 않음

### 차집합 (DIFFERNCE, -)

- 튜플의 차집합을 구하는 연산
- **카디널리티** : 빼기 당하는 릴레이션의 카디널리티보다 크지 않다.

### 교차곱 (CARTESIAN PRODUCT, x)

- 튜플의 순서쌍을 구하는 연산
- **디그리** : 두 릴레이션의 디그리를 더한 것과 같음
- **카디널리티** : 두 릴레이션의 카디널리티를 곱한 것과 같음

## 관계 해석 (Relation Calculus)

E.F. Codd씨가 수학의 Predicate Calculus(술어 해석)에 기반을 두고 관계 데이터베이스를 위해 제안

- 관계 데이터의 연산을 표현하는 방법
- 원하는 정보가 무엇이라는 것만 정의하는 **비절차적 특성**을 가짐
- 관계 대수와 처리 기능과 능력면에서 동등하고 관계대수 식을 관계해석으로 표현 가능
- 질의어로 표현
- **종류** : 튜플 관계해석, 도메인 관계해석
- **주요 논리 기호** :

|기호|구성요소|설명|
|--|---|-----|
|∀|전칭 정량자|가능한 모든 튜플에 대하여 (For All)|
|∃|존재 전량자|하나라도 일치하는 튜플이 있음 (There Exists)|
