# 🙆‍♂️식별자
---
간단하고 쉽게 표현하면 `변수명`입니다.
Java에서 식별자는 변수뿐만 아니라 상수, 메소드, 배열, 문자열, 사용자가 정의하는 클래스나 메소드 등을 구분할 수 있는 이름입니다.


## 01. 식별자 사용 원칙

01. 식별자는 문자, 숫자, 특수문자( _ or $ )로 구성될 수 있습니다.
02. 식별자의 첫 문자는 문자나 특수문자로 시작할 수 있습니다. (숫자는 안됩니다.)
03. 예약어를 식별자로 사용할 수 없습니다.
04. true, false, null은 식별자로 사용할 수 없습니다.
05. 식별자는 길이 제한이 없습니다.
06. 같은 문자의 대소문자는 서로 다른 식별자로 취급합니다. 


## 02. 식별자 관행

01. 클래스 이름의 첫 글자는 대문자로 작성합니다.
02. 메소드, 변수, 배열, 문자열의 첫 글자는 소문자로 작성합니다.
03. 상수의 이름은 모두 대문자로 작성합니다.


# 🙋‍♂️자료형
---

## 01. 자료형이란?

`자료형`이란 변수가 가질 수 있는 `값의 형태` 입니다.


## 02. 종류

자료형의 종류는 `기본 자료형`과 `참조 자료형`으로 나눠집니다.
**기본 자료형**에는 `정수형`, `실수형`, `논리형`이 있고
**참조 자로형**에는 `클래스 형`, `인터페이스 형`, `배열 형`, `열거 형`이 있습니다. 

특징으로는 `기본 자료형`은 `값`을 갖고 
`참조 자료형`은 `주소`를 갖습니다.


### 02-1. 정수형

정수형으로는 `byte`, `short`, `int`, `long`, `char`가 있습니다.
그런데 주로 수를 나타내는 정수형은 `int`를 사용하고
문자형은 `char`를 사용합니다.

`char`는 문자를 하나만 담을 수 있답니다.

```java
char ch1 = 'a';
char ch2 = 97;
char ch3 = 1100001;
char ch4 = 0x61;
char ch5 = 0x0061;
```


### 02-2. 실수형

실수형은 부호와 지수부분, 가수부분으로 구성돼있습니다.
크기에 따라서 `float`를 사용하거나 `double`를 사용합니다. 보통 `double`를 사용하고 
`float`를 사용하려면 값 뒤에 **f**를 붙여줘야 합니다.


```java
float f = 3.14f;
double dd = 3.14;
double dd = 3.14d;
```


### 02-3. 논리형

논리형 변수는 `boolean`으로 선언할 수 있고
`true`와 `false`를 갖습니다.

```java
boolean t = true;
boolean f = false;
```


### 02-4. 상수형

상수형 변수는 선언 후 값을 바꿀 수 없는 특징이 있습니다.
`final`을 사용해서 선언하고 변수명은 모두 대문자로 사용하는 관례가 있습니다.
변수만 선언하고 값을 다른데서 할당하는 경우도 있습니다.

```java
final int TEN = 10;
final boolean T;

T = true
```
