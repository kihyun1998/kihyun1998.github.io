# **Section 177.** 네트워크 관련 신기술
---

## 지능형 초연결망

과기정통부 주관 사업으로 급격하게 증가하는 데이터 트래픽을 효과적으로 수용하기 위해 시행되는 정부 주관 사업

- 망 전체에 소프트웨어 정의 기술(SDE)를 적용
- 사물 인터넷, 클라우드, 빅데이터, 5G를 효율적으로 사용하기 위한 목적이 있음


---


## 소프트웨어 정의 기술 (SDE; Software-Defined Everything)

네트워크, 데이터센터 등에서 소유한 자원을 가상화하여 개별 사용자에게 제공하고 중앙에서는 통합적으로 제어가 가능한 기술

- `소프트웨어 정의 네트워킹` : 
    - ( SDN; Software Defined Networking )
    - 네트워크를 컴퓨터처럼 모델링하여 여러 사용자가 네트워킹을 가상화하여 제어하고 관리하는 네트워크
    - 하드웨어 네트워크 체계보다 효율적임
    - 기존 네트워크 영향 안주고 경로 수정을 통해 문제 처리

- `소프트웨어 정의 데이터 센터`
    - ( SDDC; Software Defined Data Center )
    - 데이터 센터의 모든 자원 가상화 > 인력 개입 X

- `소프트웨어 정의 스토리지`
    - ( SDS; Software-Defined Storage )
    - 물리적인 데이터 스토리지를 가상화

## IoT ( Internet of Things )

실세계와 가상 세계의 다양한 사물들을 인터넷으로 연결

- 유비쿼터스 공간 구현을 위한 기술
- M2M의 개념을 인터넷으로 확장
- **기술** : 스마트 센싱, 유무선 통신 및 네트워크 인프라, 사물 인터넷 인터페이스
- 개방형 아키텍처가 필요함 > 정보 보안 기술 적용 중요

### 관련 용어

- `M2M` ( Machine to Machine )
    - 기계와 기계 사이 무선 통신
    - 변압기 원격 감시, 버스 운행 시스템 등..
- `메시 네트워크` ( Mesh Network )
    - 차세대 이동통신 등 특수 목적 네트워크
    - 그물망과 같이 유기적 연결하여 모든 구간에 동일 무선망 구성하여 안정적 사용
    - 대규모 디바이스의 네트워크 생성에 최적화된 네트워크
- `와이선` ( Wi-Sun )
    - 저전력 장거리 통신 기술
    - 낮은 지연속도, 메시 네트워크 기반 확장 > 검침 분야에 유용
- `UWB` ( Ultra WideBand )
    - 짧은 거리, 많은 양, 낮은 전력을 위한 기술
    - 블루투스와 비교되는 기수
    - 땅 속, 벽 뒤 전파 가능 > 인명 구조 가능
- `피코넷` ( PICONET )
    - 여러 독립 통신장치가 블루투스, UWB 사용해서 형성하는 무선 네트워크
    - 수십 미터 이내 형성
- `USN` ( Ubiquitous Sensor Network )
    - 센서로 정보를 무선 수집 네트워크
    - RFID 태그로 정보 탐지
- `SON` ( Self Organizing Network )
    - 자동 구성 네트워크
    - 주변 셀과 협력, 자체적 치유도 가능
- `저전력 블루투스 기술` (BLE; Bluetooth Low Energy)
    - 일반 블루투스와 동일한 주파수 대역이지만 대기 상태일 때 절전 모드
    - 시계, 장난감, 비콘 등 극소형 사물 인터넷에 적합
- `NFC` ( Near Field Communication )
    - 고주파를 이용한 근거리 양방향 RFID 일종 통신
    - 13.56MHz, 10cm 최고 424Kbps 지원

## 클라우드 컴퓨팅 ( Cloud Computing )

중앙 컴퓨터 두고 인터넷으로 접근 가능 환경

- 중앙 컴퓨터는 복수의 데이터 센터를 가상화 기술로 통합한 대형 데이터 센터
- 사용자는 키보드 마우스 등과, 통신 포트만 연결하면 사용 가능
- **클라우드 컴퓨팅 vs 그리드 컴퓨팅**
    - **그리드 컴퓨텅** > 수많은 컴퓨터를 하나의 컴퓨터 처럼 묶어 분산 처리 방식( 대규모 연산 )
    - **클라우드 컴퓨팅** > 중앙에 대형 컴퓨터 센터에서 빌려주는 형식


### 관련 용어

- `모바일 클라우드 컴퓨팅` ( MCC; Mobile Cloud Computing )
    - 모바일 기기로 클라우드 컴퓨팅 자원 공유
- `인터클라우드 컴퓨팅` ( Inter-Cloud Computing )
    - 여러 클라우드 서비스나 자원 연결하는 기술
    - 대등 접속 ( Peering ) : 직접 연계
    - 연합 ( Federation ) : 동적 자원 할당 지원하여 논리적으로 하나의 서비스 제공
    - 중개 ( Intermediary ) : 서비스 제공자간 직, 간접적인 자원 연계 또는 서비스 제공가를 통한 중개 서비스 제공
- `클라우드 기반 HSM` ( Cloud-based Hardware Security Module )
    - 암호화 키의 생성, 저장, 처리를 클라우드 기반으로 하는 보안 기기
    - 클라우드에 인증서 저장
    - 키 생성 > 하드웨어적이라 소프트웨어적인 키 보안 취약점 무시
- `파스-타` ( PaaS-TA )
    - 소프트웨어 개발 환경을 위한 개방형 클라우드 컴퓨팅
    - 관리 환경, 실행 환경, 개발 환경, 서비스 환경 운영 환경

## 기타 용어

- `징` ( Zing )
    - 10cm 이내 3.5Gbps인 초고속 NFC
    - 광고나 키오스크에 접목
- `NDN` ( Named Data Networking )
    - 콘텐츠 자체의 정보와 라우터 기능만으로 데이터 전송 수행
    - 콘텐츠 중심 네트워킹과 같은 개념
    - Hash Table 기반 P2P 시스템과 같이 정보와 라우터 기능만으로 목적지 확정
- `NGN` ( Next Generation Network )
    - ITU-T 개발, 유선망 차세대 통신망
    - 완전한 이동성, 모든 서비스를 패킷으로 압축하여 전송하는 것이 목표
- `올(all)-IP`
    - 유선 전화, 무선망, 패킷 데이터 망 등 통신망을 모두 IP 기반의 망으로 통합한 차세대 네트워크
- `WBAN` ( Wireless Body Area Network )
    - 웨어러블 네트워킹 기술
- `GIS` ( Geographic Information System )
    - 지리적 자료 시스템, 위성을 통해 정보 제공
- `애드 혹 네트워크` ( Ad-Hoc Network )
    - 재난 현장 등에서 모바일 호스트만을 이용한 네트워크
    - 기지국 필요없어서 빠른 망 구성과 저렴한 비용
- `네트워크 슬라이싱` ( Network Slicing )
    - 5G 핵심 기술
    - 하나의 물리적인 네트워크를 다수의 가상 네트워크로 분리하여 맞춤형 서비스 제공
- `파장 분할 다중화` ( WDM; Wavelength Division Multiplexing )
    - 광섬유 통신 기술
    - 복수 신호 보내서 여러 단말기가 동시에 통신 회선을 사용할 수 있도록 하는 기술
- `개방형 링크드 데이터` (LOD, Linked Open Data)
    - 웹상에 공개된 연계 데이터
    - 웹상 존재 데이터에 URL 부여하여 식별 및 연결
- `SSO` ( Single Sign On )
    - 한 번 로그인하면 모든 사이트 이용 가능
    - 불편함 해소, 마케팅 극대화
- `스마트 그리드` (Smart Grid)
    - 전력 IT
    - 전력선 기반으로 통신, 효율적인 에너지 관리 가능
