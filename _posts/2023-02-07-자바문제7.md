# 🙋‍♂️ 숫자 비교하기
---
## 문제 설명

정수 `num1`과 `num2`가 주어질 때 `num1`을 `num2`로 나눈 나머지를 return하도록 solution함수를 완성하시오.

## 문제 코드

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = -1;
        return answer;
    }
}
```

자바에서 나머지 연산자가 `%`인 점을 이용해서 해결할 수 있습니다.

## 정답 코드

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = -1;
        answer = num1 % num2;
        return answer;
    }
}
```
