# **Section 134.** 연산자
---

## 산술 연산자

산술 계산에 이용되는 연산자

|연산자|의미|비고|
|--|----|
|+|덧셈|
|-|뺄셈|
|*|곱셈|
|/|나눗셈|
|%|나머지|
|++|증가 연산자|
|--|감소 연산자|

> 증감 연산자는 `전치`인 경우 `후치`인 경우가 있습니다.
>> `전치` :증감 시킨 후 변수를 연산에 사용 함  ex) ++a
>> `후치` : 연산에 사용한 후 증감 시킴   ex) a++



## 관계 연산자

관계를 비교해서 참 거짓을 결과로 얻는 연산자

|연산자|의미|
|--|----|
|==|같다|
|!=|같지 않다|
|>|크다|
|>=|크거나 같다|
|<|작다|
|<=|작거나 같다|


## 비트 연산자

비트별 연산하여 결과 얻는 연산자

|연산자|의미|설명|
|--|---|-----|
|&|and|모든 비트가 1일 때만 1|
|^|xor|모든 비트가 같으면 0, 하나라도 다르면 1|
|&#124;|or|모든 비트 중 한 비트라도 1이면 1|
|~|not|각 비트의 부정, 0이면 1, 1이면 0|
|<<|왼쪽 시프트|비트를 왼쪽으로 이동|
|>>|오른쪽 시프트|비트를 오른쪽으로 이동|


## 논리 연산자

논리 값을 연산하는 연산자

|연산자|의미|설명|
|--|---|----|
|!|not|부정|
|&&|and|모두 참이면 참|
|&#124;&#124;|or|하나라도 참이면 참|


## 대입 연산자

연산식을 간단하게 입력할 수 있는 연산자

|연산자|예|설명|
|--|----|----|
|+=|a += 1|a = a + 1|
|-=|a -= 1|a = a - 1|
|*=|a *= 1|a = a * 1|
|/=|a /= 1|a = a / 1|
|%=|a %= 1|a = a % 1|
|<<=|a <<= 1|a = a << 1|
|>>=|a >>= 1|a = a >> 1|


## 조건 연산자

조건에 따라 서로 다른 수식을 수행

```java
조건 ? 연산1 : 연산2;
```

> 조건이 참이면 연산1, 거짓이면 연산2 수행


## 기타 연산자

|연산자|의미|
|--|-------|
|sizeof|자료형의 크기 표시|
|,(콤마)| - 두 개 이상의 수식 작성하거나 변수 정의 <br> - 좌에서 우 순으로 수행됨, 순서 연산자로 부름|
|(자료형)| - 다른 자료형으로 변환할 때 사용 cast 연산자라고도 함|


## 연산자 우선순위

1. 단항 연산자 ( !, ~, ++, sizeof ... 등)
2. 이항 연산자 ( 산술 연산자 > 시프트 연산자 > 관계 연산자 > 비트 연산자 > 논리 연산자)
3. 삼항 연산자 ( ? : )
4. 대입 연산자 ( =, += .. 등)
5. 순서 연산자 ( 콤마 )
