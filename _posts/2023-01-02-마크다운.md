## 00 파일 명
---
파일명 형식은 yyyy-mm-dd-title.md 형식을 유지해줘야 합니다.

## 01 italic과 bold 그리고 글자 색

## 02 코드 박스
---
```markdown
```(프로그래밍 언어 구분)

코드

```(닫아주는 백틱)
```

위와 같은 형식으로 작성할 수 있습니다.

### Python
```python
a=1
b=2
print(a+b)
```

### HTML
```html
<html>
  <body>
	  <div>
		  이렇게
	  </div>
  </body>
</html>
```
