# 안녕하세요

마크다운을 통해서 글을 작성 해야해서 연습을 좀 해볼게요~ <br>


## 00 파일 명
---
파일명 형식은 yyyy-mm-dd-title.md 형식을 유지해줘야 합니다.

## 01 italic과 bold
---
이텔릭은  *이렇게*  또는  _이렇게_ *(별표)와 _(언더바)를 이용해서 표현합니다.<br>
볼드체는  **이렇게**  또는 __이렇게__ *(별표)와 _(언더바)를 두 개씩 사용해서 표현합니다.

```markdown
이텔릭은  *이렇게*  또는  _이렇게_ *(별표)와 _(언더바)를 이용해서 표현합니다.<br>

```


## 02 코드 박스
---
```markdown
```(프로그래밍 언어 구분)

코드

```(닫아주는 백틱)
```

위와 같은 형식으로 작성할 수 있습니다.<br>

### Python
```python
a=1
b=2
print(a+b)
```

### HTML
```html
<html>
  <body>
	  <div>
		  이렇게
	  </div>
  </body>
</html>
```
