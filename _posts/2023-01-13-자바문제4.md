# 🙋‍♂️두 수의 곱
---
## 문제 설명

정수 `num1`과 `num2`가 주어질 때 `num1`을 `num2`로 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

## 문제 코드

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        return answer;
    }
}
```

간단하게 `answer`에 `num1`과 `num2`를 뺀 값을 할당해주면 됩니다.
자바에서 뺄셈 연산자는 `-` 입니다.

## 정답 코드

```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = 0;
        answer = num1 - num2;
        return answer;
    }
}
```
